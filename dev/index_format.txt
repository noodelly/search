<page> Specifies a new webpage, inside of the tags is the extrapolated title </page>
<index> Specifies an index </index>
<rating> Specifies a rating for a webpage </rating>
<url> Specifies the URL of the index </url>


Example:
<index>
	<page>A webpage</page>
	<url> https://awebpage.com </url>
	<rating>90</rating>
</index>
